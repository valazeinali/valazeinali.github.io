1. (Based on Rice 6.3) Let Xbar be the average of a sample of n independent standard normal RVs.¶

(a) Determine c such that P(|X| < c) = 0.5. Solve for c as a function of n.

(b) Using only R *norm() functions for the standard normal distribution, compute the exact value of c for n = 5,...100 and visualize as a plot of c vs. n.

(c) If the variance was not known, how would you solve the problem and what additional piece of information would you need to get an exact answer?

(d) If the n RVs are independent and have the same distribution with expectation 0 and variance 1 but the exact distribution is not known, how would you approach the problem?

a) \begin{equation} P(|\bar{X} \leqslant c|) = P(-c \leqslant \bar{X} \leqslant c) = P(\bar{X} \leqslant c) - P(\bar{X} \leqslant -c) \\ = P(\bar{X} \leqslant c) - [1-P(\bar{X} \leqslant c)] = 2P(\bar{X} \leqslant c)-1 = 2F_{\bar{X}}c-1 \\ P(\bar{X} \leqslant c) - 1 = 2\phi(\frac{c-0}{\sqrt{\frac{1}{n}}}) - 1 = 2\phi{\sqrt{n}*c}-1 = .5 \\ 2\phi(\sqrt{n}*c) = 1.5 = \phi(\sqrt{n}*c) = .75 \\ = \sqrt{n}*c = \phi^{-1}(.75) \\ \therefore c = \frac{.674}{\sqrt{n}} \end{equation}

7 (Based on Rice 7.16) True or false? Justify your answers.¶

(a) The center of a 95% confidence interval for the population mean is a random variable.

(b) A 95% confidence interval for contains the sample mean with probability 0.95.

(c) A 95% confidence interval contains 95% of the population.

a) This is true due to the nature of the formula. Since the formula includes a sample mean, which is random due to the value beeing prone to change from sample to sample. In addition, this is also true for a sample standard deviation. Thus, since these two metrics are random and confidence intervals are contructed with these parametes, the center and spread of a confidence interval will change from sample to sample.

b) This is false because a 95% confidence interval for pop. mean will contain the population mean with certainty 1, not .95.

c) This assumption is false, a 95% confidence interval refers to 95/100 times that the interval from samples will capture the true mean, not the population. It does not need to capture the set fraction of the population.
